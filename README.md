## First public project

### [Codex Danger Gate](https://github.com/wilsongpt1/codex-danger-gate)

An open-source Windows plugin that adds an independent human confirmation step before supported high-risk Codex Agent actions run. It helps reduce the risk of an AI agent accidentally executing a destructive command while keeping the project's security boundaries and limitations explicit.

[View the project](https://github.com/wilsongpt1/codex-danger-gate) · [Download the latest release](https://github.com/wilsongpt1/codex-danger-gate/releases/latest)

Codex Danger Gate is the first of my tools that I'm sharing publicly. I plan to share more practical AI tools, implementation patterns, and experiments shaped by the requirements and lessons of enterprise environments.
